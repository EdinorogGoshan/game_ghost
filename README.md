# game_ghost

Нейросеть: DeepSeek (DeepSeek AI)
Время разработки: ~9 часов

Основные возможности:
- Управление персонажем (призраком) с физикой платформера
- 3 уровня сложности с прогрессивной сложностью
- Система жизней и очков
- Различные типы врагов с патрулированием и AI
- Собираемые предметы (огоньки) для получения очков
- Опасные платформы со шипами (сверху и снизу)
- Анимации для всех персонажей и эффектов
- Полноценный игровой цикл с меню паузы и game over

Технические особенности:
- Чистый JavaScript (без сторонних библиотек)
- Canvas для отрисовки графики
- Объектно-ориентированная архитектура
- Асинхронная загрузка ресурсов
- Раздельная система столкновений
- Режим отладки с визуализацией хитбоксов

Инструкциями по запуску:
- открыть страницу в браузере, игра начнется сразу при загрузке страницы;
- ДЛЯ ОТЛАДКИ: клавиша F1 показывает хитбоксы, а клавишами 1, 2, 3 (не numpud) можно пеерключаться между уровнями
